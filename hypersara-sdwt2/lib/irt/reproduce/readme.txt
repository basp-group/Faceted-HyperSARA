This directory contains m-files and some data for "reproducible research,"
i.e., the code associated with publications from Jeff Fessler's group.


muckley-15-fpm/

Matthew J Muckley  Douglas C Noll  J A Fessler
Fast parallel MR image reconstruction via B1-based, adaptive restart,
iterative soft thresholding algorithms (BARISTA)
IEEE Tr. Medical Imaging, 34(2):578-88, Feb. 2015
http://dx.doi.org/10.1109/TMI.2014.2363034


ramani-12-asb/

Sathish Ramani  J A Fessler
A splitting-based iterative algorithm for accelerated statistical X-ray CT reconstruction
IEEE Tr. Medical Imaging, 31(3):677-88, Mar. 2012
http://dx.doi.org/10.1109/TMI.2011.2175233


sutton-04-dfm/

Bradley P Sutton  Douglas C Noll  J A Fessler
Dynamic field map estimation using a spiral-in / spiral-out acquisition
MRM 51(6):1194-204, June 2004
http://dx.doi.org/10.1002/mrm.20079
